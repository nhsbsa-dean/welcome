Getting set up

When you are finished introduction stuff and ready to be working on the project I have asked Christian Enfield to help get you set up with OHS on your machine because there's a few things to sort out for it.

I'm sure he will explain what each application is and what we need to have running.

The main applications you will use are:

  • OHS-UI (OHS admin application)
  • CRA-UI (Get healthcare cover for traveling abroad application)
  • Entitlements (stored user information, this mainly just runs in the background)

You will have access to these which Christian will link you to, when you have a Gitlab account created for you.

Set up instructions are on confluence too, ohs dev setup.
